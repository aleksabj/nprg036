Folder queries containing:
Files query-1-1.jq, query-1-2.jq, query-2-1.jq, ... with executable queries. The first number identifies the data file the query should be run against. The second number identifies the queries on one file.
File readme.txt in UTF-8 explaining the meaning of the individual queries.

Query descriptions:

query-1-1.jq:
Extracts all events and lists the people participating in each event based on matching IRIs.

query-1-2.jq:
Extracts all events together with their location and start date.

query-2-1.jq:
Lists all actors together with the number of films they appear in, sorted by that count.

query-2-2.jq:
Extracts all award winners together with the award, prize amount, and the movie the award refers to.
